"Brighton" Times Website

This was my first project in the Udacity Nanodegree courses. I learned a lot about responsive website design such using media queries to dictate the styling of a website based on parameters such as minimum and maximum screen sizes. This allows for a full website experience regardless of device used to view the website. I also learned about the benefits of using a framework like bootstrap (which I used to create this current webpage in fact!)

Responsive Images Blog

This was my second project in the Udacity Nanodegree courses. I learned alot about the importance of images in a modern website, and the importance of using the right sizes for the right devices (through the use of srcset for example). I learned about alternatives to external images such as Icon Fonts, inline SVGs and IMGs. I also learned that latency is the biggest factor in a website's performance, and to always consider this when creating a website.
